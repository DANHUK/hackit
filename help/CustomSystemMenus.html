Custom System Menus

Hack-It allows you to add certain useful actions to the System Menu of any top-level application window. It currently supports four different actions:

  • Hiding a window
  • Making a window topmost, or making a window not topmost
  • Adding or removing a tray icon for a window
  • Setting the priority of the process associated with the window

These options are enabled in the Settings dialog, under the Window Control tab.

Note: Certain windows in the system, such as DOS prompts, are not represented the same way as other windows, and do not have standard System Menus. These windows will unfortunately not receive the custom actions.
