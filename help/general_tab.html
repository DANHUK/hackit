General tab

The general tab contains settings for configuring the general operation of Hack-It. Most of these settings change how Hack-It operates as an application, and how some of the inner workings of Hack-It behave.

  • Hide Hack-It on Startup - Sets whether the Hack-It main window will be displayed on start-up. Default is off (Unchecked).

  • Show Splash Screen On Startup - Sets whether the splash screen is shown when Hack-It is started. Default is on (Checked).

  • Auto Functions Enabled – If you do not use Autofunctions, and would like to turn them off, this is the place to do it. Default is on (Checked).

  • Auto Functions Affect Children – You can specify that Autofunctions should also apply to (match) child windows. Default is off (Unchecked).

  • Hack-It Window Always On Top – If you would like the Hack-It main window to have topmost status, set this option. Default is off (Unchecked).

  • Hack-It Priority – Select one of the four options to set the process priority of Hack-It. Higher settings reserve more processor time for Hack-It. Default is Normal.

  • Hack-It Font – You can choose what font is used in the Window List. Default is the same font your system has designated for windows.
