Hack-It v4.0 Help
Illegal Windows Tab

Illegal Window List: This is a list of Illegal Window specifications. Illegal Window specs describe windows that are not to be shown in the window list, nor affected by automatic functions. The specifications in this list prevent Hack-It from responding to or listing particular windows on your system. Examples:

  • No window owned by Hack-It is ever shown on the window list

  • Many people prevent certain windows from appearing because they clutter the list, such as menus, tool tips, and message boxes

  • Some windows do not behave correctly when Hack-It queries them for information (example: VB6 will crash if Hack-It is running and you debug a program), and are good candidates to be ignored completely.

Add: Press ‘Add’ to bring up the Illegal Window Entry dialog and create a new Illegal Window specification. Changes take effect immediately after the add.

Edit: Press ‘Edit’ to bring up the Illegal Window Entry dialog and make change to an existing Illegal Window specification. You must select a specification to edit. Changes take place immediately after leaving the Illegal Window Entry dialog.

Delete: Press ‘Delete’ to delete the currently selected Illegal Window specification from the list. You must select a specification to delete. Changes take effect immediately.
