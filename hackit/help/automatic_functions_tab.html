Hack-It v4.0 Help
Automatic Functions Tab

Automatic Function List: All of your Automatic Functions are listed here by name. Each function must have a unique name.

Function Information: A summary of the selected Automatic Function is shown here, based on what actions and criteria you specified for it.

Add: Pressing ‘Add’ will bring up the Add AutoEntry dialog, allowing you to create a new Automatic Function. Only one Automatic Function can be added at a time. Changes to Hack-It behavior are immediate following the add.

Edit: Pressing ‘Edit’ will bring up the Edit Auto Entry dialog, allowing you to modify the currently selected Automatic Function. You must select a function from the list before pressing ‘Edit’. Changes to Hack-It behavior are immediate after modifications are complete.

Delete: Pressing ‘Delete’ will delete the currently selected Automatic Function from the list. You will not be prompted to confirm your delete. Changes to Hack-It behavior are immediate.
